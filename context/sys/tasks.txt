Here's a breakdown for creating a REST API with user authentication:

---

1.  **GOAL**
    Build a RESTful API that allows users to register, log in, and access protected resources using JWT-based authentication.

2.  **STEPS**

    *   **Step 1: Project Setup & Basic Server**
        - [ ] Initialize a new Node.js project (`npm init -y`).
        - [ ] Install core dependencies: `express`, `dotenv`, `cors`.
        - [ ] Create `server.js` (or `app.js`) and set up a basic Express server.
        - [ ] Configure `dotenv` for environment variables.
        - [ ] Define a root (`/`) GET endpoint that returns a simple "API is running" message.
        - [ ] Start the server and verify it's accessible.

    *   **Step 2: Database Setup & Connection (e.g., MongoDB with Mongoose)**
        - [ ] Install database driver/ORM: `mongoose`.
        - [ ] Configure database connection in `server.js` using `process.env.MONGO_URI
